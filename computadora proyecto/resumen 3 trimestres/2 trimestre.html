EL DESARROLLO DE LA TECNOLOGIA
La tecnología ha sido pleado para satisfacer de manera sustentable las necesidades de los seres humanos sin poner en riesgo al medio ambiente y así ofrecer a las nuevas generaciones mejor condiciones ambientales.
DESARROLLO TECNOLOGICO SUSTENTABLE
Es fundamental evitar una agotamiento de los recursos y prevenir daños significativos al medio ambiente ya que para utilizarlos debemos mantener un equilibrio adecuado entre el uso de los recursos y la conservación del medio ambiente para el no dañar al medio ambiente.
VISION PROSPECTIVA DE LA TECNOLOGIA
Las nuevas tecnologías deben De ser responsables y respetuosos hacia el medio ambiente y los recursos ya que deben de estar apoyando a la reducción del consumo de agua la utilización de la energía renovable y la conservación del equilibrio ecológico ya que cada forma de vida en el planeta depende de los seres humanos.
LAS 3 ERRES
Hay muchas maneras y estrategias que podemos usar para cuidar el medio ambiente una de ellas es la regla de la 3 erres que son reducir, reusar y reciclar la regla en las 3 erres hace que se reduzca la producción de objetos que dañen al medio ambiente y también hace que se vuelvan a utilizar para darle una segunda vida con el mismo uso u otro diferente.

PROGRAMACION
La programación es. Un acto de programar se refiere a organizar una secuencia de pasos ordenados a seguir para hacer cierta cosa. Este termino puede utilizarse en muchos contextos es común hablar de programación a la hora de organizar una salida, las vacaciones, o la lista de programas con sus dias y horarios de emisión.
ALGORITMO
Un algoritmo es un conjunto de instrucciones o reglas de bebidas ordenadas infinitas que permite típicamente solucionar un problema o procesar datos y llevar a cabo otras tareas o actividades. siguiendo unos pasos sucesivos se llega a un estado final y se obtiene una solución o resultado a ese problema.
DIAGRAMA DE FLUJO
Un diagrama de flujo es una representación gráfica y secuencial de un proceso o flujo de trabajo con todas las tareas de actividades principales necesarias para lograr un objetivo establecido el diagrama de flujo utiliza las figuras geométricas. Estos diagrama de flujos usan formas especiales para representar diferentes tipos de acciones o pasos en proceso también usan líneas y flechas que muestran la secuencia de los pasos y las relaciones entre ellos.
TIPOS DE DATOS
Existen varios tipos de datos que determinan el tipo de información que puede almacenarse en él a continuación se muestran algunos de estos: Entero este tipo de datos se utiliza para almacenar números enteros, es decir, números sin parte decimal, 1,24,11,101, etc. Real se utiliza para almacenar números con parte decimal 3.5, 8.04,44.45, etc. Se utiliza para almacenar una secuencia de caracteres como una palabra o una frase: Buen día, a, verdadero, etc. Lógico se utiliza para almacenar valores bolianos es decir verdadero o falso. Estos datos nos ayudan cuando queremos realizar algo y no sabemos en qué información se almacena cada uno de estos.
OPERADORES RELACIONALES
Son símbolos que permiten comparar dos valores y determinar su relación verdadero o falsa en pseint los operadores relacionales más comunes son los siguientes: (= - igual a ),(<> o != - distinto de),(> - mayor que),(< - menor que),(>= - mayor o igual que),(= - menor o igual que).
